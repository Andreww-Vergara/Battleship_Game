# Battleship_Game

A terminal-based implementation of the classic Battleship game developed from scratch using Python.

The project applies Object-Oriented Programming principles to model players, ships, and game mechanics. 
Matrix-based structures are used to manage the game board, while custom algorithms handle ship placement, 
turn management, hit detection, and victory conditions.

Technologies Used:
Python
Concepts Applied:
Object-Oriented Programming (OOP)
Matrix manipulation
Algorithm design
Game logic implementation
Key Features:
Interactive console gameplay
Turn-based mechanics
Ship placement system
Hit and miss detection
Win condition validation
